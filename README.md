# Brighter : Edit your photo with 40+ filters and 10 other tools.

## App Features :
- **Filters** - Brighter app has 40+ cool filters.
- **Exposure**
- **Contrast**
- **Crop**
- **Orientation**
- **Vintage**
- **Round Corner**
- **Saturation**
- **Sharpness**
- **Warmth**
- **Draw**
- **Stickers**
